# antd-mobile[cnode]


实现功能：
  1. 登录
  2. 获取主题
  3. 主题详情
  4. 新建主题
  5. 编辑主题
  6. 收藏主题
  7. 取消主题
  8. 用户所收藏的主题
  9. 新建评论
  10. 为评论点赞
  11. 用户详情
  12. 消息列表（标记全部已读）
  

### 下载源码 ###
    git clone https://github.com/renzhilong/cnode-webapp.git
### 安装依赖 ###
    npm install
### 开发测试 ###
    npm start
### 发布代码 ###
    npm run build
